### IPEX
These tools allow you to make an FTP connection with your server and transfer files to another created by developer and hacker A.X.E.L
### INSTALLATION
- Just Follow steps
```
$ git clone https://github.com/7AXEL/CLOUDFTP
```
```
$ cd CLOUDFTP
$ bash install.sh
```
### RUN
```
$ python main.py
```
### DEPENDENCISE

**`CLOUDFTP`** requires following programs to run properly - 
- `openjdk-17`
- `ruby`
- `python3`
- `mplayer`
> Supported Platform : **`Termux`**, **`Ubuntu/Debian/Kali/Parrot`**, **`Arch Linux/Manjaro`**, **`Fedora`**
